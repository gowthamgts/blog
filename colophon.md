### Devices
#### Desktop
Since my preferred development environment is macOS for quite sometime (even though I'm regretting about upgrading to macOS 26), I have a Mac Mini M2 Pro 2023 which is my daily driver and also a 2019 Intel based Macbook Pro that's just gathering dust since I bought the Mini.

I also have custom built a PC that's intended solely for gaming. I have written about it [here]({% post_url blog/2024-12-14-first-pc-build %}).

Both of the above devices were connected to a Dell 27-inch (U2720Q) monitor, a Keychron keyboard and an MX master 3s mouse.

#### Phone
My daily driver is an iPhone 16 along with an Apple watch 10. I earlier had an OnePlus 8t which was a great phone that survived multiple crashes. I usually alternate between an android and an iOS device to avoid the apple ecosystem, but this time I'm okay paying the apple tax for the time-being.

#### Servers
I do have one raspberry pi 4B at home which [hosts]({% post_url blog/2025-12-18-self-hosting-setup-2025 %}) some of my services that's running raspbian os. I do have some cloud servers and all of them run Ubuntu.

### Network
For now, my home network is only one device - an [Asus AC-59u V2](https://www.asus.com/networking-iot-servers/wifi-routers/asus-wifi-routers/rt-ac59u-v2/) router which comfortably handles all our devices with a 400Mbps connection from our ISP. The raspberry pi is the only device that's connected over LAN.

### Editors
VS code is my goto editor these days. I developed a taste for Intellij based IDEs, but I can live without them. I use Cursor at work and it saves me time due to the tab driven development. I started using claude code for some of my personal projects lately.

In all my editors, I have configured VIM motions. I also try out neovim from time to time, but I go back to the other editors when it gets in my way (mostly because I forgot some keybindings).
