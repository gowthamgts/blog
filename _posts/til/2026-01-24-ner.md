While researching on how to cluster some document contents together in an ML pipeline for a pet project, I came across [NER (Named Entity Recognition)](https://en.wikipedia.org/wiki/Named-entity_recognition). Apparently, it's a technique in NLP which can be used to identify named entities like persons, places, date, organizations, etc.

In python you can use libraries like [spaCy](https://spacy.io/), [BERT](https://github.com/google-research/bert), etc to do this or you can also use LLM to extract this info as shown [here](https://huggingface.co/datasets/polyglots/Tamil-NER-Transliterated/viewer).
